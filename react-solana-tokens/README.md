# React Solana Tokens

Simple examples of ReactJS app connecting to Solana blockchain using browser wallet and sending spl-token.

## How to Run
1. git clone the project
2. cd react-solana-tokens
3. copy .env.example as .env
4. fill .env variables
5. npm install
6. npm run dev
